## Professional Portfolio

This is my personal portfolio, built to showcase my work, projects, and approach to finance, data science, and risk.

### Overview
The site highlights my experience, technical projects, and core focus areas, with an emphasis on clean design and user experience.

### Tech Stack
- React / Next.js  
- Tailwind CSS  
- Framer Motion (animations)  
- EmailJS (contact form)  

### Development
- Built locally using VS Code  
- Version controlled with Git  
- Deployed on Vercel  

### Focus
This portfolio reflects how I approach my work:
- Systems thinking  
- Risk and uncertainty  
- Data-driven decision-making
